# Pulse - pulse oximeter shop website

**In this project I practiced:**
* JavaScript
* BEM methodology
* SCSS
* Animate.css
* PHP Mailer
* Gulp

**Demo:**
[Pulse website](https://olga-volga.github.io/pulse/dist)

**To run this project locally:**
* clone with `git clone https://github.com/olga-volga/pulse.git`
* `cd pulse`
* `npm install`
* `gulp`
